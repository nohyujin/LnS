# Phase 3: 분석 코드 작성

## 작업 배경

CS 상담 데이터를 분석하여 오안내율, 평균 상담 시간(AHT)을 계산하고, 임계치를 초과하는 이슈를 자동으로 탐지하는 핵심 분석 로직을 구현합니다. 이 모듈은 대시보드의 데이터 처리 엔진 역할을 합니다.

## 작업 내용

### 1. 데이터 로딩 모듈 (`data_loader.py`)
- CSV/Excel 파일 읽기 함수
- 데이터 전처리 함수 (타입 변환, 정제)
- 데이터 유효성 검사 함수

### 2. 분석 로직 구현 (`analyzer.py`)
- 오안내율 계산 함수:
  - 전체 오안내율
  - 상담유형별 오안내율
  - 시간대별 오안내율
- 평균 상담 시간(AHT) 계산 함수:
  - 전체 평균 AHT
  - 상담유형별 평균 AHT
  - 성공/오안내별 평균 AHT

### 3. 경고 로직 구현
- 임계치 설정 (오안내율 5%)
- 임계치 초과 감지 함수
- 우선순위 도출 알고리즘:
  - 오안내율 가중치
  - 발생 빈도 가중치
  - 영향도 계산

### 4. 성과 예측 로직
- 교육 효과 예측 함수
- 예상 AHT 단축 시간 계산
- ROI 계산 로직

- [ ] 우선순위 알고리즘이 구현됨
