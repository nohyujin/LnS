# Phase 4: UI 구성

## 작업 배경

Streamlit을 사용하여 사용자가 데이터를 업로드하고, 분석 결과를 시각적으로 확인할 수 있는 대시보드를 구현합니다. 오안내율 차트, 경고 알림, 우선순위 리스트 등을 포함한 직관적인 UI를 제공합니다.

## 작업 내용

### 1. Streamlit 기본 설정
- `src/app.py` 메인 파일 생성
- 페이지 설정 (제목, 아이콘, 레이아웃)
- 사이드바 구성

### 2. 데이터 업로드 인터페이스
- 파일 업로더 위젯 구현 (CSV/Excel 지원)
- 업로드된 데이터 미리보기
- 데이터 통계 요약 표시

### 3. 대시보드 메인 화면
- 전체 통계 카드 (총 상담 건수, 평균 AHT, 전체 오안내율)
- 오안내율 차트:
  - 시계열 차트 (날짜별 추이)
  - 카테고리별 바 차트
  - 상담유형별 파이 차트
- AHT 차트:
  - 상담유형별 평균 AHT
  - 성공/오안내 비교 차트

### 4. 경고 알림 패널
- Andon 경고 메시지 표시 (`st.warning()` 또는 `st.error()`)
- 임계치 초과 항목 강조 표시
- 경고 수준별 색상 구분:
  - 빨간색: 긴급 (10% 이상)
  - 주황색: 주의 (5-10%)
  - 노란색: 관찰 (3-5%)

### 5. 우선순위 리스트
- 교육 시급도 순위 테이블
- 각 항목별 상세 정보 (상담유형, 오안내율, 발생 건수, 예상 개선 효과)

### 6. 성과 예측 대시보드
- 교육 실시 시 예상 효과 표시
- 예상 AHT 단축 시간
- 예상 비용 절감 효과
- 개선 목표 시각화

## 인수 조건

- [ ] `src/app.py` 파일이 생성됨
- [ ] `streamlit run src/app.py` 명령으로 앱이 실행됨
- [ ] 파일 업로드 기능이 정상 작동함
- [ ] 업로드된 데이터가 정상적으로 표시됨
- [ ] 모든 차트가 정상적으로 렌더링됨
- [ ] 경고 알림이 임계치에 따라 정확히 표시됨
- [ ] 우선순위 리스트가 정렬되어 표시됨
- [ ] 성과 예측 정보가 표시됨
- [ ] 반응형 레이아웃이 적용됨
