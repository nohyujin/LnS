# Phase 5: 교육 가이드 생성 기능

## 작업 배경

탐지된 이슈에 대한 교육 가이드를 자동으로 생성하는 기능을 구현합니다. Markdown 템플릿을 기반으로 동적 콘텐츠를 생성하고, PDF 또는 HTML 형식으로 출력하여 즉시 배포 가능한 교육 자료를 제공합니다.

## 작업 내용

### 1. 템플릿 시스템 구축
- Markdown 템플릿 파일 생성 (`templates/template_guide.md`)
- 템플릿 구조 정의:
  - 문제 상황 섹션
  - 변경된 정책 섹션
  - 올바른 응대 스크립트 섹션
  - 참고 자료 섹션

### 2. 동적 콘텐츠 생성
- `src/guide_generator.py` 생성
- 템플릿에 데이터 삽입 함수
- 이슈별 맞춤 콘텐츠 생성 로직

### 3. 파일 출력 기능
- HTML 변환 함수
- PDF 변환 함수 (선택)
- 파일명 자동 생성 (날짜_이슈유형.pdf)
- 로컬 폴더 저장 기능 (`output/` 디렉토리)

### 4. UI 통합
- 가이드 생성 버튼 추가
- 생성 완료 메시지 표시
- 다운로드 링크 제공
- 생성된 가이드 미리보기

## 인수 조건

- [ ] HTML 파일이 정상적으로 생성됨
- [ ] PDF 파일이 정상적으로 생성됨 (선택)
