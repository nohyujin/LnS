로컬 환경에서 최소한의 비용으로 구현하면서도, 쿠팡 ANDON팀이 요구하는 **"데이터 기반의 기민한 교육 솔루션"** 역량을 가장 확실하게 보여줄 수 있는 MVP(Minimum Viable Product) 프로젝트를 제안합니다.

이 프로젝트는 별도의 유료 서버나 클라우드 구독 없이, 본인의 PC(로컬) 내에서 **Python**과 **Streamlit**을 활용해 **"CS 교육 이슈 자동 탐지 및 가이드 생성기"**를 만드는 것입니다.

# 프로젝트명: [Local-Andon] 실시간 CS 결함 탐지 및 교육 대응 툴

이 프로젝트는 "현장에서 문제가 발생했을 때(Andon), 데이터를 분석하여 즉시 교육 솔루션을 배포한다"는 시나리오를 로컬 프로그램으로 구현한 것입니다.

## 1. 핵심 기술 스택 (100% 무료 & 로컬)

* **데이터 소스:** **Excel / CSV** (가상의 CS 상담 로그 데이터)
* **분석 및 로직:** **Python** (Pandas를 활용한 데이터 처리)
* **대시보드/UI:** **Streamlit** (Python만으로 웹 기반 UI를 로컬에서 실행 가능한 라이브러리)
* **콘텐츠 템플릿:** **Markdown / HTML** (로컬에 저장된 가이드라인 양식)

---

## 2. MVP 핵심 기능 (3-Step)

### Step 1: 가상 CS 데이터 시뮬레이션 (Excel)

실제 데이터를 구할 수 없으므로, 쿠팡에서 발생할 법한 데이터를 엑셀로 생성합니다.

* **컬럼 구성:** `상담ID`, `상담유형(배송/결제/오배송)`, `상담결과(성공/오안내)`, `통화시간(AHT)`, `VOC 키워드`
* **핵심 로직:** 특정 유형(예: '신선식품 반품')에서 '오안내' 빈도가 급증하는 데이터를 일부러 심어둡니다.

### Step 2: Andon 교육 트리거 대시보드 (Streamlit)

로컬에서 Python 파일을 실행하면 웹 브라우저에 다음과 같은 화면이 뜹니다.

* **이슈 탐지:** 오안내율이 설정된 임계치(예: 5%)를 넘으면 빨간색 **'Andon 경고'** 표시.
* **우선순위 도출:** "현재 가장 교육이 시급한 항목은 '쿠팡이츠 배달 지연 대응'입니다"라는 인사이트 자동 출력.
* **성과 예측:** "해당 교육 실시 시 예상 상담 시간   단축 기대"와 같은 수식 결과 노출.

### Step 3: 즉시 배포용 교육 가이드 자동 생성

* **기능:** 버튼 하나를 누르면 해당 이슈에 맞는 **'원페이지(One-page) 교육 가이드'**가 PDF나 HTML 파일로 로컬 폴더에 자동 생성됩니다.
* **내용:** [문제 상황 - 변경된 정책 - 올바른 응대 스크립트]가 포함된 표준 양식.

---

## 3. 구현 로드맵 (초단기)

1. **데이터 준비 (1시간):** Excel에 약 100줄 정도의 가상 상담 데이터를 만듭니다.
2. **분석 코드 작성 (2시간):** Python의 Pandas 라이브러리로 '오안내율'과 '평균 상담 시간'을 계산하는 코드를 짭니다.
3. **UI 구성 (2시간):** Streamlit을 사용해 차트와 경고 메시지를 화면에 띄웁니다.
* *코드 예시:* `st.warning("배송 이슈 오안내 급증! 즉시 교육이 필요합니다.")`


4. **결과물 정리:** 로컬에서 구동되는 화면을 **영상으로 캡처**하고, 이 툴이 어떻게 'Andon'의 철학을 교육에 녹여냈는지 포트폴리오로 정리합니다.

---

## 4. 이 프로젝트가 쿠팡 채용 담당자에게 주는 메시지

* **"나는 데이터를 직접 다룰 줄 압니다":** 단순히 교육 운영만 하는 사람이 아니라, Python/Excel을 활용해 문제를 스스로 정의할 줄 압니다.
* **"나는 효율성에 집착합니다":** 매번 수동으로 분석하는 게 아니라, 로컬에서 자동화된 툴을 만들어 교육 기획의 '속도'를 높였습니다.
* **"나는 ANDON의 본질을 이해합니다":** 현장의 결함을 발견하는 즉시 교육으로 연결하는 프로세스를 시스템적으로 설계했습니다.
