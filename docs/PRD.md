# PRD: Local-Andon - 실시간 CS 결함 탐지 및 교육 대응 툴

## 1. 프로젝트 개요

### 1.1 프로젝트 배경
쿠팡 ANDON팀이 요구하는 **"데이터 기반의 기민한 교육 솔루션"** 역량을 보여주기 위한 MVP(Minimum Viable Product) 프로젝트입니다. 현장에서 문제가 발생했을 때(Andon), 데이터를 분석하여 즉시 교육 솔루션을 배포하는 시나리오를 로컬 환경에서 구현합니다.

### 1.2 프로젝트 목표
- 로컬 환경에서 최소한의 비용으로 구현
- CS 교육 이슈를 자동으로 탐지하고 가이드를 생성하는 시스템 개발
- 데이터 분석 역량과 교육 솔루션 설계 능력 입증

### 1.3 프로젝트 범위
- **포함:** CS 상담 데이터 분석, 이슈 탐지, 교육 가이드 자동 생성
- **제외:** 실제 서버 배포, 실시간 데이터 연동, 사용자 인증 시스템

---

## 2. 기술 스택

### 2.1 핵심 기술 (100% 무료 & 로컬)

| 구분 | 기술 | 용도 |
|------|------|------|
| 데이터 소스 | Excel / CSV | 가상의 CS 상담 로그 데이터 |
| 분석 및 로직 | Python (Pandas) | 데이터 처리 및 분석 |
| 대시보드/UI | Streamlit | 웹 기반 UI (로컬 실행) |
| 콘텐츠 템플릿 | Markdown / HTML | 교육 가이드라인 양식 |

## 3. 기능 요구사항

### 3.1 Step 1: 가상 CS 데이터 시뮬레이션

#### 3.1.1 데이터 구조
실제 쿠팡에서 발생할 법한 CS 상담 데이터를 Excel로 생성합니다.

**필수 컬럼:**
- `상담ID`: 고유 식별자
- `상담유형`: 배송/결제/오배송 등
- `상담결과`: 성공/오안내
- `통화시간(AHT)`: Average Handle Time
- `VOC 키워드`: 고객의 소리 키워드

#### 3.1.2 데이터 특성
- 특정 유형(예: '신선식품 반품')에서 '오안내' 빈도가 급증하는 패턴 포함
- 최소 100개 이상의 샘플 데이터

### 3.2 Step 2: Andon 교육 트리거 대시보드

#### 3.2.1 이슈 탐지 기능
- **입력:** CSV/Excel 형식의 CS 상담 데이터
- **처리:** 오안내율 계산 및 임계치 비교
- **출력:** 임계치(예: 5%) 초과 시 빨간색 'Andon 경고' 표시

#### 3.2.2 우선순위 도출
- 현재 가장 교육이 시급한 항목 자동 식별
- 예시: "현재 가장 교육이 시급한 항목은 '쿠팡이츠 배달 지연 대응'입니다"

#### 3.2.3 성과 예측
- 교육 실시 시 예상되는 상담 시간 단축 효과 계산
- 데이터 기반 ROI 예측 제공

#### 3.2.4 UI 구성요소
- 실시간 데이터 업로드 기능
- 오안내율 차트 (시계열/카테고리별)
- 경고 알림 패널
- 우선순위 리스트
- 성과 예측 대시보드

### 3.3 Step 3: 즉시 배포용 교육 가이드 자동 생성

#### 3.3.1 가이드 생성 기능
- **트리거:** 버튼 클릭
- **출력 형식:** PDF 또는 HTML 파일
- **저장 위치:** 로컬 폴더 (자동 생성)

#### 3.3.2 가이드 내용 구성
표준 양식에 따라 다음 내용을 포함:
1. **문제 상황:** 탐지된 이슈 설명
2. **변경된 정책:** 관련 정책 또는 프로세스 변경사항
3. **올바른 응대 스크립트:** 실제 사용 가능한 상담 스크립트

#### 3.3.3 템플릿 시스템
- Markdown 기반 템플릿
- 동적 데이터 삽입 (이슈 유형, 통계 등)
- 일관된 포맷 유지

## 7. 핵심 가치 제안

이 프로젝트를 통해 다음을 입증합니다:

### 7.1 데이터 활용 능력
> **"나는 데이터를 직접 다룰 줄 압니다"**
> 
> 단순히 교육 운영만 하는 사람이 아니라, Python/Excel을 활용해 문제를 스스로 정의하고 해결할 수 있습니다.

### 7.2 효율성 추구
> **"나는 효율성에 집착합니다"**
> 
> 매번 수동으로 분석하는 게 아니라, 로컬에서 자동화된 툴을 만들어 교육 기획의 '속도'를 높였습니다.

### 7.3 ANDON 철학 이해
> **"나는 ANDON의 본질을 이해합니다"**
> 
> 현장의 결함을 발견하는 즉시 교육으로 연결하는 프로세스를 시스템적으로 설계했습니다.
