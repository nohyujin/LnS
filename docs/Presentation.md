# Local-Andon
## 실시간 CS 결함 탐지 및 교육 대응 툴

**데이터 기반의 기민한 교육 솔루션**

---

## 📋 프로젝트 개요

### 프로젝트 배경
- 쿠팡 ANDON팀의 **"데이터 기반의 기민한 교육 솔루션"** 역량 입증
- 현장 문제 발생 시 즉시 교육 솔루션 배포
- 로컬 환경에서 최소 비용으로 구현

### 프로젝트 목표
✅ 로컬 환경에서 최소한의 비용으로 구현
✅ CS 교육 이슈 자동 탐지 및 가이드 생성
✅ 데이터 분석 역량과 교육 솔루션 설계 능력 입증

---

## 🎯 핵심 가치 제안

### 1. 데이터 활용 능력
> "나는 데이터를 직접 다룰 줄 압니다"

Python/Excel을 활용해 문제를 스스로 정의하고 해결

### 2. 효율성 추구
> "나는 효율성에 집착합니다"

자동화된 툴로 교육 기획의 속도 향상

### 3. ANDON 철학 이해
> "나는 ANDON의 본질을 이해합니다"

현장 결함 발견 즉시 교육으로 연결하는 시스템적 설계

---

## 🛠 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| **데이터 소스** | Excel / CSV | 가상 CS 상담 로그 데이터 |
| **분석 및 로직** | Python (Pandas) | 데이터 처리 및 분석 |
| **대시보드/UI** | Streamlit | 웹 기반 UI (로컬 실행) |
| **콘텐츠 템플릿** | Markdown / HTML | 교육 가이드라인 양식 |

### 💰 100% 무료 & 로컬

---

## 🚀 핵심 기능 Overview

### Step 1: 가상 CS 데이터 시뮬레이션
실제 쿠팡 CS 상담과 유사한 데이터 생성

### Step 2: Andon 교육 트리거 대시보드
이슈 탐지, 우선순위 도출, 성과 예측

### Step 3: 교육 가이드 자동 생성
즉시 배포 가능한 교육 가이드 생성

---

## 📊 Step 1: 가상 CS 데이터 시뮬레이션

### 데이터 구조

**필수 컬럼:**
- `상담ID` - 고유 식별자
- `상담유형` - 배송/결제/오배송 등
- `상담결과` - 성공/오안내
- `통화시간(AHT)` - Average Handle Time
- `VOC 키워드` - 고객의 소리 키워드

### 데이터 특성
- 최소 100개 이상의 샘플 데이터
- 특정 유형에서 오안내 빈도 급증 패턴 포함

---

## 🎛 Step 2-1: 이슈 탐지 기능

### 입력
CSV/Excel 형식의 CS 상담 데이터

### 처리
오안내율 계산 및 임계치 비교

### 출력
임계치(5%) 초과 시 **빨간색 'Andon 경고'** 표시

---

## 📈 Step 2-2: 우선순위 도출

### 자동 식별
현재 가장 교육이 시급한 항목 자동 식별

### 예시 출력
> "현재 가장 교육이 시급한 항목은 
> **'쿠팡이츠 배달 지연 대응'**입니다"

---

## 💡 Step 2-3: 성과 예측

### 예측 내용
- 교육 실시 시 예상 상담 시간 단축 효과
- 데이터 기반 ROI 예측

### UI 구성요소
- 실시간 데이터 업로드
- 오안내율 차트 (시계열/카테고리별)
- 경고 알림 패널
- 우선순위 리스트
- 성과 예측 대시보드

---

## 📝 Step 3: 교육 가이드 자동 생성

### 가이드 생성 기능
- **트리거:** 버튼 클릭
- **출력 형식:** PDF 또는 HTML
- **저장 위치:** 로컬 폴더 (자동 생성)

### 가이드 내용 구성
1. **문제 상황** - 탐지된 이슈 설명
2. **변경된 정책** - 관련 정책/프로세스 변경사항
3. **올바른 응대 스크립트** - 실제 사용 가능한 상담 스크립트

---
