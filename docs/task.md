## Phase 2: 데이터 준비 (1시간)

### 2.1 데이터 구조 설계
- [ ] Excel 파일 생성 (`cs_data.xlsx`)
- [ ] 필수 컬럼 정의
  - [ ] 상담ID (고유 식별자)
  - [ ] 상담유형 (배송/결제/오배송/반품/교환 등)
  - [ ] 상담결과 (성공/오안내)
  - [ ] 통화시간(AHT) (초 단위)
  - [ ] VOC 키워드
  - [ ] 상담일시 (timestamp)
  - [ ] 상담사ID

### 2.2 샘플 데이터 생성
- [ ] 최소 100개 이상의 샘플 데이터 생성
- [ ] 정상 케이스 데이터 (70%)
- [ ] 오안내 케이스 데이터 (30%)
- [ ] 특정 이슈 패턴 삽입
  - [ ] 신선식품 반품 오안내 급증 패턴
  - [ ] 쿠팡이츠 배달 지연 오안내 패턴
  - [ ] 결제 오류 오안내 패턴

### 2.3 데이터 검증
- [ ] 데이터 무결성 확인
- [ ] 컬럼별 데이터 타입 확인
- [ ] 결측치 확인 및 처리
